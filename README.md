# 🎓 Student Placement Prediction System

A complete end-to-end machine learning project that predicts student placement outcomes using academic and skill-based features.

---

## 🚀 Features

- 🔮 Real-time placement prediction using ML model
- 📊 Interactive Streamlit dashboard
- 📁 Data preprocessing & EDA (Jupyter Notebooks)
- 🗄️ MySQL database integration
- 📈 Power BI dashboard for visualization
- ⚡ Fast and user-friendly UI

---

## 🧠 Machine Learning Model

- Model Used: Decision Tree Classifier
- Input Features:
  - IQ Score
  - Academic Performance
  - Internship Experience
  - CGPA (Categorized)
  - Communication Skills
  - Projects

- Output:
  - Likely to be Placed
  - Not Likely to be Placed

- Model also provides prediction confidence.
